Dólar Elite | Terminal Profesional
Una aplicación web moderna para monitorear cotizaciones del dólar argentino en tiempo real con interfaz profesional y efectos visuales de profundidad.

Características
6 Tipos de Dólar: Oficial, Blue, Tarjeta, MEP, CCL y Cripto
Conversión en Tiempo Real: Ingresa un monto en ARS y conviértelo automáticamente a USD
Gráficos Interactivos: Mini-gráficos de área con ApexCharts mostrando el historial de precios
Modo Oscuro/Claro: Toggle de tema con persistencia en localStorage
Actualización Automática: Precios actualizados cada 30 segundos
Sistema de Notificaciones: Alertas de error con reintentos automáticos
Diseño Responsivo: Adaptado para desktop, tablet y móvil
Efectos de Profundidad: Sombras, gradientes y animaciones suaves
Tecnologías
HTML5
CSS3 (con gradientes, animaciones y efectos modernos)
JavaScript (ES6+)
ApexCharts (gráficos interactivos)
API: dolarapi.com
Instalación
Clona o descarga el repositorio
Abre index.html en tu navegador
No requiere instalación de dependencias (usa CDN para ApexCharts)
Uso
Ver Cotizaciones: Los precios se actualizan automáticamente
Convertir ARS a USD: Escribe el monto en pesos en el campo de input
Cambiar Tema: Click en el botón "Cambiar Modo" arriba a la derecha
Ver Historial: Los mini-gráficos muestran los últimos 10 precios
